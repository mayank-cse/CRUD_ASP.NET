# CRUD_ASP.NET_C#

## Prerequirements

* Visual Studio
* .NET SDK
* SQL Server

## How To Run

* Open solution in Visual Studio
* Set .Web project as Startup Project and build the project.
* Change the Connection String in Web.Config
* Run the application.
